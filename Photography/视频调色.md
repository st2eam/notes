# 视频调色

## 色彩校正 vs 调色

| | 色彩校正（Correction） | 调色（Grading） |
|---|---|---|
| 目的 | 还原真实色彩 | 创造风格化色调 |
| 顺序 | 先做 | 后做 |
| 操作 | 白平衡、曝光、对比度校正 | 色调映射、风格化处理 |
| 目标 | 各镜头色彩统一、准确 | 营造情绪和视觉风格 |

## 示波器（Scopes）

调色时不能只靠眼睛，需要示波器提供客观数据。

### 波形图（Waveform）

显示画面从左到右的亮度分布。

- Y 轴：亮度值（0 = 纯黑，100 = 纯白）
- 广播安全范围：0~100 IRE
- 波形底部堆积 → 欠曝
- 波形顶部削平 → 过曝

### 矢量图（Vectorscope）

显示画面的色相和饱和度分布。

- 中心 = 无色彩（灰/白/黑）
- 距中心越远 = 饱和度越高
- 角度方向 = 色相
- 肤色参考线：10 点钟方向（黄色和红色之间）

### RGB 分量

分别显示红绿蓝三个通道的分布，用于精确校正白平衡。

## 色彩校正流程

### 第一步：曝光校正

1. 调整**黑色色阶**，让最暗处接近 0
2. 调整**白色色阶**，让最亮处接近 100
3. 调整**整体曝光**，让中间调合适

### 第二步：白平衡校正

使用画面中的中性灰/白色区域为参考：

- 色温偏暖（偏黄）→ 降低色温
- 色温偏冷（偏蓝）→ 提高色温
- 偏绿 → 增加品红
- 偏品红 → 增加绿色

### 第三步：对比度

- 调整 Lift（暗部）、Gamma（中间调）、Gain（高光）
- 或使用曲线工具精确控制

### 第四步：饱和度

- 校正阶段保持自然饱和度
- 修正过饱和或欠饱和的颜色
- 注意肤色不要过于饱和

## 调色技巧

### 色轮调色

使用三向色轮（Lift / Gamma / Gain）控制不同亮度区间的色调偏移：

| 色轮 | 控制区域 | 常用操作 |
|------|---------|---------|
| Lift（暗部） | 阴影区域 | 加青/蓝营造冷调阴影 |
| Gamma（中间调） | 画面主体 | 控制整体色调倾向 |
| Gain（高光） | 明亮区域 | 加暖色营造温暖高光 |

### 经典色调参考

#### 电影青橙色调（Teal & Orange）

- 阴影偏青蓝（Teal）
- 高光/肤色偏橙暖
- 互补色对比，视觉冲击力强

#### 胶片色调

- 阴影提起（不到纯黑），高光压下（不到纯白）
- 降低整体饱和度
- 暗部偏绿/青，高光偏暖
- 添加少量颗粒

#### 高对比冷调

- 高对比度
- 整体偏冷蓝
- 降低肤色饱和度
- 适合悬疑、科幻题材

#### 日系清新

- 低对比度
- 提阴影，压高光
- 整体偏青绿
- 低饱和度

### LUT 的使用

LUT（Look-Up Table）是预设的色彩映射表：

- **技术 LUT** — 用于 Log 素材转 Rec.709（还原正常色彩）
- **创意 LUT** — 应用特定风格（如胶片模拟）

使用建议：

1. 先做色彩校正，再应用 LUT
2. LUT 强度建议 60%~80%，不要 100% 直接套用
3. 应用后微调曝光和饱和度
4. 不同光照条件下同一 LUT 效果不同，需要逐镜头调整

## 导出色彩设置

| 用途 | 色彩空间 | 说明 |
|------|---------|------|
| 网络视频 | Rec.709 | 标准色彩空间，兼容性最好 |
| HDR 视频 | Rec.2020 / HLG / PQ | 更广色域和动态范围 |
| 影院 | DCI-P3 | 电影院放映标准 |
| 交付母版 | 与拍摄时色彩空间一致 | 保留最大后期空间 |
