# 摄影基础

## 曝光三要素

曝光由三个参数共同决定，它们构成"曝光三角"：

### 光圈（Aperture）

用 f 值表示，如 f/1.4、f/2.8、f/5.6、f/11。

- **f 值越小** → 光圈越大 → 进光量越多 → 景深越浅（背景虚化）
- **f 值越大** → 光圈越小 → 进光量越少 → 景深越深（前后都清晰）

| 场景 | 推荐光圈 |
|------|---------|
| 人像特写 | f/1.4 - f/2.8 |
| 街拍 | f/4 - f/5.6 |
| 风光 | f/8 - f/11 |
| 星芒效果 | f/16 - f/22 |

### 快门速度（Shutter Speed）

传感器接收光线的时间长度。

- **高速快门**（1/1000s+）→ 凝固运动瞬间
- **中速快门**（1/60 - 1/250s）→ 日常拍摄
- **慢速快门**（1s+）→ 光轨、丝绢流水

> 手持安全快门 ≈ 1/焦距。例如 50mm 镜头，安全快门约 1/50s。

### ISO 感光度

传感器对光线的敏感程度。

- **低 ISO**（100-400）→ 画质细腻，噪点少
- **高 ISO**（1600+）→ 暗光可用，但噪点增加

## 测光模式

| 模式 | 说明 | 适用场景 |
|------|------|---------|
| 评价测光 | 综合整个画面亮度 | 大多数场景 |
| 中央重点 | 侧重画面中央区域 | 人像 |
| 点测光 | 只测量对焦点附近 | 逆光、高反差 |

## 构图法则

### 三分法

将画面用两横两竖分成九宫格，将主体放在交叉点或线条上。

### 引导线

利用道路、河流、栏杆等线条引导视线到主体。

### 框架构图

用门框、窗户、树枝等自然元素框住主体，增加层次感。

### 对称与倒影

建筑、水面等场景适合对称构图，营造平衡感。

## 白平衡

白平衡决定照片的色温倾向：

| 色温（K） | 效果 | 场景 |
|-----------|------|------|
| 3000K | 暖黄调 | 室内暖光、日落 |
| 5500K | 自然中性 | 日光 |
| 7000K+ | 冷蓝调 | 阴天、蓝调时刻 |

## RAW vs JPEG

| 特性 | RAW | JPEG |
|------|-----|------|
| 文件大小 | 大（20-60MB） | 小（3-10MB） |
| 后期空间 | 极大 | 有限 |
| 动态范围 | 12-14 bit | 8 bit |
| 直出可用 | 需后期处理 | 可直接使用 |

建议：重要拍摄用 RAW + JPEG 双格式保存。
