# 后期处理

## Lightroom 基础工作流

### 1. 镜头校正

导入后第一步开启镜头校正，修正畸变和暗角。

### 2. 基本调整

按顺序调整以下参数：

1. **白平衡** — 校正或营造色温氛围
2. **曝光** — 调整整体亮度
3. **对比度** — 增加画面层次
4. **高光 / 阴影** — 压高光提阴影，找回细节
5. **白色色阶 / 黑色色阶** — 确定画面亮度范围

### 3. 色调曲线

S 型曲线增加对比度：提亮高光区、压暗阴影区。

```
     亮
  ──/──
 /
/──────
     暗
```

胶片感：抬起曲线左下角（阴影不是纯黑），压下右上角（高光不是纯白）。

### 4. HSL 调色

分别调整各颜色的色相、饱和度、明亮度：

- **肤色优化** — 降低橙色饱和度，提高橙色明亮度
- **天空增强** — 提高蓝色饱和度，降低蓝色明亮度
- **绿植调整** — 黄绿色相偏移，控制植物色调

### 5. 锐化与降噪

- 锐化量 40-80，半径 1.0-1.2
- 按住 Alt 拖动蒙版滑块，只锐化边缘
- 高 ISO 照片适当增加明亮度降噪

## 色彩理论

### 互补色

色轮上对立的颜色，对比强烈：

- 蓝 ↔ 橙（日落天空与海洋）
- 青 ↔ 红（电影感色调）
- 绿 ↔ 品红

### 类似色

色轮上相邻的颜色，和谐统一：

- 黄 + 橙 + 红（秋天色调）
- 蓝 + 青 + 绿（清新自然）

## 常见风格参考

| 风格 | 特点 |
|------|------|
| 日系清新 | 低对比、低饱和、偏青绿、提阴影 |
| 胶片复古 | 褪色黑、偏暖、颗粒感、低饱和 |
| 赛博朋克 | 高对比、霓虹色、青橙分离 |
| 高级灰 | 低饱和、中性色调、柔和对比 |
| 电影感 | 青橙互补、宽银幕比例、暗角 |
