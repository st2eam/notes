### 常见的 HTML 事件

下面是一些常见的 HTML 事件：

| 事件          | 描述                |
| ----------- | ----------------- |
| onchange    | HTML 元素已被改变       |
| onclick     | 用户点击了 HTML 元素     |
| onmouseover | 用户把鼠标移动到 HTML 元素上 |
| onmouseout  | 用户把鼠标移开 HTML 元素   |
| onkeydown   | 用户按下键盘按键          |
| onload      | 浏览器已经完成页面加载       |

更完整的列表：[W3School JavaScript 参考手册 HTML DOM 事件。](https://www.w3school.com.cn/jsref/dom_obj_event.asp)
